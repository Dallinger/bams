# Active model selection

## Installation
```
pip install bams
```

## Usage
```Python
import bams
import oracle

learner = bams.learners.ActiveLearner(
    query_strategy=bams.strategies.BALD,
    budget=10,
    base_kernels=["PER", "LIN", "SE", "LG"],
    max_depth=2,
    dims=3,
)

while learner.budget > 0:
  x = learner.next_query()
  y = learner.query(oracle, x)
  learner.update(x, y)
```
